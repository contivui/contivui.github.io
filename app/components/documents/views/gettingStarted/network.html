Getting Started with Contiv Network

There are several ways to try out Contiv Network. All of these ways fall into two categories:

  • Use our Vagrant scripts to set up a small-scale virtual environment on an single machine.
  • Install Contiv Network on two or more bare-metal or virtual servers.

On a Single Machine

You can use our Vagrant script to get up and running fast on a single Linux or OS X machine. Vagrant is a utility for quickly and consistently setting up virtual machines (VMs).

  • Try out Contiv Network with Docker Swarm.
  • Try out Contiv Network with Kubernetes.
  • Try out Contiv Network with Mesos.
  • Try out Contiv Network with Nomad.

On Multiple Servers

You can install Contiv on two or more baremetal nodes or VMs.

  • Install Contiv Network with Docker Swarm.
  • Install Contiv Network with Kubernetes.
  • Install Contiv Network in L3 BGP mode.
  • Install Contiv Network with Cisco ACI*.

*For more information about using Contiv Network with Cisco Application Centric Infrastructure (ACI), contact Cisco Systems.
